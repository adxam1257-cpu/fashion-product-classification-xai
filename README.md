# fashion-product-classification-xai
# Fashion Product Classification using CNN with Explainable AI

## 📌 Project Overview
This project implements a Convolutional Neural Network (CNN) to classify fashion products using the Fashion MNIST dataset. It also includes Explainable AI (Grad-CAM) to visualize model decisions.

---

## 📊 Dataset
- Fashion MNIST (70,000 images)
- 10 classes of clothing items
- Link: https://www.kaggle.com/datasets/zalando-research/fashionmnist

---

## 🧠 Model Architecture
- Convolutional Layers
- Max Pooling Layers
- Flatten Layer
- Dense Layers

---

## 🎯 Performance
- Test Accuracy: ~91%

---

## 🔍 Explainable AI
Grad-CAM is used to show which parts of the image influence the model’s decision.

---

## 📁 Files
- Fashion_Product_Classification.ipynb (executed notebook)
- README.md
- report (Overleaf PDF)

---

## 🚀 How to Run
1. Open the notebook in Google Colab
2. Run all cells sequentially
3. Ensure dataset is loaded correctly
4. View predictions and Grad-CAM visualization results
